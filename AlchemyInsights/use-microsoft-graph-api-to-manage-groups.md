# <a name="use-microsoft-graph-api-to-manage-groups"></a>Използване на API Graph Microsoft за управление на групи

Групите са колекции от потребители и други главници, които споделят достъп до ресурсите Услуги на Microsoft или във вашето приложение. Microsoft Graph предоставя API, които можете да използвате за създаване и управление на различни типове групи и функционалност на групи според вашия сценарий. Всички операции, свързани с групата, в Microsoft Graph изискват съгласие от администратор. За повече информация вижте Работа [с групи в Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
