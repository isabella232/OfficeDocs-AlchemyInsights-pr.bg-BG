# <a name="how-to-type-special-characters-on-a-mac"></a>Как да въведете специални знаци на Mac

За да въведете специални знаци на вашия Mac, отворете **менюто** Специални знаци с клавишната комбинация ⌘ **command**  +  **Control**  +  **Space**.

Бърз пряк път за @ също е **ALT**  +  **64**.
